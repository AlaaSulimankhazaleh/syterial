Settings
<tabs>
  • animated (Default: false): Apply a CSS fade transition to the tab panes.
  • justified (Default: false): Make the tabs stretch at equal sizes to span the entire width.
  • scrollable (Default: false): Make the tab content scrollable if specific height is exceeded. See scrollable-height.
  • scrollable-height (Default: 300): Set the max-height-breakpoint for scrollable content. The unit is 'px'.
<pane>
  • title (Default: 'Pane'): Set the title for this pane.
  • icon (Default: none): Set the icon next to the title of this pane. (CSS-class for the icon required)
